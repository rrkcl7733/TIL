### 2. username max length

> The `max_length` should be sufficient for many use cases. If you need a longer length, please use a [custom user model](https://docs.djangoproject.com/en/4.0/topics/auth/customizing/#specifying-custom-user-model). If you use MySQL with the `utf8mb4` encoding (recommended for proper Unicode support), specify at most `max_length=191` because MySQL can only create unique indexes with 191 characters in that case by default.

![image-20220412162006641](C:\Users\USER\AppData\Roaming\Typora\typora-user-images\image-20220412162006641.png)

```
2가지 문제점
1. 함수 내의 logout 함수는 
from django.contrib.auth import logout
에서 가져온 것일텐데 
정의된 함수이름과 중복된다.
from django.contrib.auth import logout as auth_logout  로 별칭을 지어서 해결.

2. 로그인된 사용자에 한해 auth_logout 해야한다.
그래서 로그인 되었는지 물어보자

if request.user.is_authenticated:
        auth_logout(request)
```
